Widget.PIM.AddressBookItem
class AddressBookItem
extends Object


AddressBookItem represents a PIM Contact. A contact consists of values associated with attributes. Its possible values are "addressBookItemId", "fullName", "mobilePhone", "homePhone", "workPhone", "eMail", "company", "title", "address".
