captureImage

String captureImage(<String> fileName,<Boolean> lowRes)
    Capture still image and save to a file.

    e.g.
        Widget.Multimedia.Camera.onCameraCaptured = myCallBack;
        Widget.Multimedia.Camera.captureImage("/sdcard/a.jpg", false);
        function myCallBack(String fileName) {
            var v = fileName;
        }
    Parameters:
      fileName - the relative path where the captured image to be saved. If filename includes a fullPath, then image file is saved to that exact location.
      lowRes - the flag to specify if low resolution is used.
    Returns:
      the full path of the saved file. This value is identical to the file path provided by onCameraCaptured().

setWindow

void setWindow(<Object> DomObject)

    window before taking a picture or recording a video.Note:The size of Preview
    Window should larger than 32(pixel) * 24(pixel)
    e.g.

         mCamera.setWindow(Div);
    Parameters:
      DomObject - a DOM object which specifites the preview window.

startVideoCapture

void startVideoCapture(<String> fileName,<Boolean> lowRes,<Number> maxDurationSeconds,<Boolean> showDefaultControls)
    Start video capture.

    e.g.
        Widget.Multimedia.Camera.onCameraCaptured = myCallBack;
        function myCallBack(String fileName) {
            var v = fileName;
        }
        Widget.Multimedia.Camera.startVideoCapture("a.3gp", false, 30, true);
    Parameters:
      fileName - the full path where the captured image to be saved. If filename includes a fullPath, then image file is saved to that exact location.
      lowRes - the flag to specify if low resolution is used.
      maxDurationSeconds - the max capture time. The unit is in second. This field should be an interger value.
      showDefaultControls - the flag to specify whether to show default controls.

stopVideoCapture

void stopVideoCapture()
    Explicitly stop a video capture in process.

    e.g.
        Widget.Multimedia.Camera.startVideoCapture("a.3gp", false, 30, true);
        Widget.Multimedia.Camera.stopVideoCapture();
