addAddress

void addAddress(<String> type, <String> address)
    Add an address for the message.

    e.g.
        var msg = Widget.Messaging.createMessage(Widget.Messaging.MessageTypes.SMSMessage);
        msg.addAddress("destination", "10086")
    Parameters:
      type - type of the address to be added. Possible values are "destination", "cc" and "bcc".
      address - Address to be added to the destination address list.

    Parameters:

deleteAddress

void deleteAddress(<String> type, <String> address)
    Delete an address from the message's destination addresses.

    e.g.
        var msg=Widget.Messaging.createMessage(Widget.Messaging.MessageTypes.SMSMessage);
        msg.addAddress("cc", "10086")
        msg.addAddress("cc", "13800138000")
        msg.deleteAddress("cc", "10086")
    Parameters:
      type - type of the address to be added. Possible values are "destination", "cc" and "bcc".
      address - Address to be deleted from the destination address list.

deleteAttachment

void deleteAttachment(<Attachment> attachment)
    Delete an attachment from the Message.

    e.g.

        msg.addAttachment("/sdcard/sun.jpg");

        var att = msg.attachments;
        msg.deleteAttachment(att[0]);
    Parameters:
      attachment - the attachment to be delete from Message.
