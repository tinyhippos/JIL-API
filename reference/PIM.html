Widget.PIM
class PIM
extends Object


PIM is the container object for JIL PIM objects.

PIM is a sub-object of Widget. So we use PIM in this way: Widget.PIM.XXX

findAddressBookItems

void findAddressBookItems(<AddressBookItem> comparisonContact, <Number> startInx, <Number> endInx)
    Find all contacts matching the supplied comparison contact. Wildcard are supported and searches are case-insensitive.
    E.g., assigning "*smith*" to the "full_name" attribute of the comparisonContact will return all contacts having 'smith' in the name.
    Widget can specify only return a range of matched files between startInx and endInx.

    e.g.
        var contact = new Widget.PIM.AddresBookItem();
        contact.setAttributeValue("mobilePhone", "10086");
        Widget.PIM.findAddressBookItems(contact, 0, 9);
    Parameters:
      comparisonContact - A Contact containing attributes/values that are to be matched against the contact database
      startInx - an integer value to specify the index the returning search results starts from
      endInx - an integer value to specify the index to which the returning search results ends at

findCalendarItems

void findCalendarItems(<CalendarItem> itemToMatch, <Number> startInx, <Number> endInx)
    Find all calendar items matching the supplied itemToMatch. Wildcards are supported and searches are case-insensitive.
    E.g., assigning "*meeting*" to the 'eventNotes' attribute of the itemToMatch will return all calendar items having 'meeting' in the event notes.
    Widget can specify only search a range of files between startInx and endInx. This is an asynchronous function, and will invoke PIM.onCalendarItemsFound() when search completes.

    e.g.
        var calendarItem = new Widget.PIM.CalendarItem();
        calendarItem.eventName = "*Meeting";
        Widget.PIM.findCalendarItems(calendarItem, 0, 9);
    Parameters:
      itemToMatch - A calendar item containing attributes/values that are to be matched against the calendar item database
      startInx - an integer value to specify the index the returning search results starts from
      endInx - an integer value to specify the index to which the returning search results ends at
