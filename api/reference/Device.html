Direct Known Subclasses:
DeviceInfo, ApplicationTypes, File, DataNetworkInfo, DeviceStateInfo, AccountInfo, RadioInfo, PowerInfo, PositionInfo
class Device
extends Widget


Device provides access to the Device.

All the file/directory functions can manipulate entire file system, except that some methods may fail as the permission of file or directoy.

Usally, "/local" and "/sdcard" (if Micro SD card is inserted) directory can be used by most copy/delete/create functions.

copyFile

Boolean copyFile(<String> originalFile, <String> destinationFullName)
    This method copies the specified file to the path and file name specified in the destinationFullName. The new file's fullName property will be that specified by the destinationFullName value in this function.This method also applies to directories.

    e.g.
        var v = Widget.Device.copyFile("/sdcard/usr/a", "/sdcard/usr/b");
    Parameters:
      originalFile - The full name of file to be copied.
      destinationFullName - The full name of destination file.
    Returns:
      A boolean is returned indicating the success of the operation.

deleteFile

Boolean deleteFile(<String> destinationFullName)
    This method deletes the file located at the specified fullName. The full name includes the path and file name. This method also applies to directories.

    e.g.
        var v = Widget.Device.deleteFile("/sdcard/usr/a");
    Parameters:
      destinationFullName - The full name of file to be deleted.
    Returns:
      A boolean is returned indicating the success of the operation.

findFiles

void findFiles(<File> matchFile, <Number> startInx, <Number> endInx)
    This method returns an array of Files on all file systems that match the specified File. Widget can specify only search a range of files between startInx and endInx.
    This is an asynchronous function, and will invoke Device.onFilesFound() when search complete.

    e.g.
        Widget.Device.onFilesFound = myCallBack;
        var v = Widget.Device.findFiles(matchfile, 0, 9);
        function myCallBack(files) {
        }
    Parameters:
      matchFile - The File object as a match file
      startInx - an integer value to specify the index the returning search results starts from.
      endInx - an integer value to specify the index to which the returning search results ends at.

Array getAvailableApplications()
    Get array of the names of applications that can be launched.

    e.g.
        var v = Widget.Device.getAvailableApplications();
    Returns:
      An array of String of the application names. Possible value on the phone may be "Dialer", "ProgMng","Sound Recorder","Sync", "FileMgr", "Browser","Monternet", "Camera", "Picture", "Mobile TV" etc. This depends on what applications installed on phone

getDirectoryFileNames

Array getDirectoryFileNames(<String> sourceDirectory)
    Get array of names of all files in the specified directory.

    e.g.
        var v = Widget.Device.getDirectoryFileNames("/sdcard/usr/");
    Parameters:
      sourceDirectory - The directory's full name to get files.
    Returns:
      Returns an array of the names for all files found in the specified path. An empty array is returned if no files are found. A null value is returned if the specified path is not valid.

getFile

File getFile(<String> fullName)
    This provides access to a File object located at the specified full path and name. This fullName should correspond to the fullName property of the file, which includes the path and the file name. A right slash ('/') is used as the directory separator. This method also applies to directories.

    e.g.
        var v = Widget.Device.getFile("/sdcard/usr/mydata");
    Parameters:
      fullName - The full file name
    Returns:
      File object as a reference of the specified file. A null value is returned if the file is not found at the specified location;

moveFile

Boolean moveFile(<String> originalFile, <String> destinationFullName)
    This method moves the specified file to the path and file name specified in the destinationFullName. If success, the source file is removed from its original location and placed in the new location with the new file name. The new file's fullName property will be that specified by the destinationFullName value in this function.This method also applies to directories.

    e.g.
        var v = Widget.Device.moveFile("/sdcard/usr/a", "/sdcard/usr/b");
    Parameters:
      originalFile - The full name of file to be moved.
      destinationFullName - The full name of destination file.
    Returns:
      A boolean is returned indicating the success of the operation.
