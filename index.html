JIL-API by tinyhippos

JIL API change tracking reposity

This has been put up as a help resource for those developing Mobile Widgets using the JIL API. The intent of this repository is to allow developers to see the change set for each incremental release. This repository was born out of a very simple need: We need to know what changed between releases and the documentation provided by JIL is lacking in that respect. If any members of JIL come across this and would like to take over the maintenance, please contact us. We'd be happy to help or relinquish control.

Dependencies

none

Install

This is not meant for installation. This repository is for change tracking only
